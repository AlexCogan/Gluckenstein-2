Professor Gluckenstein's Infernal Escape

Summary

Professor Gluckenstein's Infernal Escape is a Text Adventure Game about escaping from a mysterious manor you find yourself trapped within. You can enter commands to move, pick up items, use items, and solve puzzles.

It's written in Javascript and HTML

Actions, Items, Rooms, and Descriptions will be stored with lists.

Seperate functions will be used to parse and execute commands.
